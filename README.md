# Store Manager

A full-stack Store Management System built with **ASP.NET Core Web API** and **React + TypeScript**.

##  Tech Stack

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- 3-Layer Architecture (N-Tier) with Repository Pattern

### Frontend
- React + TypeScript
- Vite
- TanStack Query (React Query)
- React Hook Form + Zod Validation
- Axios

## Features

- Products Management (CRUD)
- Customers Management (CRUD)
- Orders Management
- Returns Management
- Sales Tracking
- Dashboard with Analytics
